*copy command
cp - is used in Linux to copy the files and folders

1. to copy the file into folder
-> cp  filename.txt   folder
        (source)      (destination)

2. to copy file from one folder to another folder
-> cp   folder1/file.txt      folder2/
        (directory/source)   (another folder/destination)

3. to copy folder into another folder
-> cp -r  folder1   folder2     (-r recursively)
          (source)   (destination)

*move command
 
mv - it is used to move or rename file and folder in Linux

1. to move file from one folder to another

2. to rename file or folder

* file permission commands
1. ls -l = shows the file permissions
2.chmod - give the permission to the file and folder
3.chown - to change the ownership of the file
4.chgrp - change file or folder group onwnership

d r w x

d-> directory
r-> read can read a file content which in binary count  is 4
w-> write can write a file which is in binary count 2
x-> execute the file which is binary count 1
so here,
rwx   r-x   r--
│     │     │
│     │     └── Others (o)
│     └──────── Group (g)
└────────────── Owner (u)
Numeric (Octal) Relation

Each permission has a number:

r = 4
w = 2
x = 1
So:
rwx = 4+2+1 = 7

rw- = 4+2 = 6

r-x = 4+1 = 5

r-- = 4 = 4

example:
chmod 777 file.txt this gives full permission to file
